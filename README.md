# חוות השטוזים — נועה נסיכת השטוזים

משחק דפדפן סטטי בעברית. אין צורך בהתקנה, בסיס נתונים או שרת מיוחד.

## הפעלה

פתחו את `index.html` בדפדפן.

## פרסום ציבורי

העלו את כל תוכן התיקייה (כולל `assets`) לכל שירות אירוח סטטי, למשל GitHub Pages, Netlify או שרת האתר שלכם. קובץ הכניסה הוא `index.html`.

## תוכן המשחק

- 10 תחנות עם חיות שונות ומזון מתאים.
- שאלות חיבור עד 10 ושאלות ספירה עד 20.
- במקרה של טעות ניתן לנסות שוב עד להצלחה.
- שטוז הומוריסטי ייחודי לכל חיה.
- קריינות עברית אופציונלית, בהתאם לתמיכת הדפדפן.
- מוזיקת רקע מקורית בלולאה, עם כפתור הפעלה והשתקה נפרד.
- תצוגה מותאמת לטלפון, טאבלט ומחשב.
